#### 学习笔记
##### deep
监听响应式对象的属性（所有）
```
// 创建响应式对象
const user = ref({
  name: '张三',
  details: {
    age: 25,
  }
})

watch(user, (newValue, oldValue) => {
  console.log('用户信息发生变化:', newValue)
}, { deep: true })
```
精确监听某个属性
```
watch(
    () => user.value.age  // 要监听的属性
    () => {
        log('变化了')
    }
)
```
##### 模板引用
通过ref标识获取真实的dom对象或者组件实例对象
```
<script setup>
import {ref} from 'vue'

//1. 调用ref函数得到ref对象
const h1ref = ref(null)
const comref = ref(null)
<script>

<template>
//2. 通过ref标识绑定ref对象
  <h1 ref="h1ref">我是dom标签h1</h1>  //dom标签
  <TestCom ref="comref"/>  //组件
</template>
```
##### v-bind
v-bind 用于把 JavaScript 中的数据绑定到 HTML 的属性上
语法：`v-bind:属性名="变量"` 简写：`:属性名="变量"`
##### v-if条件渲染 
只会在指令的表达式返回真值时才被渲染
##### v-for 列表渲染
为了给Vue一个提示，以便它可以跟踪每个节点的标识，从而重用和重新排序现有的元素，你需要为每个元素对应的块提供一个唯一的key
`<div v-for="item in items" :key="item.id">{{item.name}}</div>`
##### v-model
表单双向绑定，用来把表单元素的值与组件数据双向绑定
组件上的v-model:
Vue 3 统一采用 modelValue + update:modelValue 事件（默认名字），实现双向绑定，也支持多个 v-model（通过修饰名）
