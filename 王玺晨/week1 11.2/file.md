## HTML学习笔记/记录

### HTML 基本结构
- 理解了 `<!DOCTYPE html>` 声明
- 掌握了 `<html>`, `<head>`, `<body>` 三大基本标签的作用
- 学会了使用 `<title>` 设置网页标题

### 常用文本标签
- 使用 `<h1>` 到 `<h6>` 定义不同级别的标题
- 使用 `<p>` 标签定义一个段落
- 使用 `<br>` 标签进行换行

### 媒体与链接标签
- 使用 `<img>` 标签插入图片，并理解了 `src` 和 `alt` 属性的重要性
- 使用 `<a>` 标签创建超链接，掌握了 `href` 属性的用法

### 列表与表格
- 使用 `<ul>` 和 `<li>` 创建无序列表
- 使用 `<table>`, `<tr>`, `<td>` 创建简单的表格

## 遇到难题与解决方法

### 困难
不太理解 `<div>` 和 `<span>` 标签的具体使用场景和区别

### 解决方法
通过查阅 MDN Web Docs 官方文档，我明白了 `<div>` 是一个块级元素，用于布局分组，而 `<span>` 是一个行内元素，用于对文本中的一部分进行样式化

### 问题
图片路径引用错误，导致图片无法在网页上显示

### 解决方法
检查了图片文件是否与 HTML 文件在同一个文件夹内，并确保 `src` 属性中的文件名和扩展名完全正确
