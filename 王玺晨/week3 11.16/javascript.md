# javascript学习
### 导入方式
#### head的内联样式
``` <head><script></script></head>```
#### body的内联样式
``` <body><script></script></body>```
#### 外联样式
``` <script src="./javascript.js"></script>```
##### 在控制台输出内容
``` console.log("") ```
##### 弹窗
``` alert("") ```
### 变量和数据类型
- var用于函数内变量
``` var x; ```
- let用于块级变量，更常用
``` let y=10; ```
- const表示常量
```const z='js'```
### 控制语句
- 条件语句：和c++一样
```if(){}else(){}```
- 循环语句：和c++一样
``` for(let i=0;i<10;i++){} ```
``` while(){}```
```break;```
```continue;```
### 函数
- 用法和c++基本相同
```function name(){}```
### javascript事件
- onClick点击 
- onfocus获取焦点
- onflur失去焦点
可以用在html标签内：
``` 
<button onClick="函数名">click</button>
```
