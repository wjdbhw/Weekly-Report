### AJAX
#### HTTP协议
- HTTP请求报文
<br>
```javascript
//请求报文
//请求行 请求方法（类型） 请求URL 请求版本（http版本）
//请求头 请求头名1：值1
//      请求头名2：值2
//      请求头名3：值3
//空行 （必须有）
//请求体 （可选择）
//
//
//响应报文
//响应行  响应版本（http版本）响应状态码（200） 响应状态文本（OK）
//响应头 响应头名1：值1
//      响应头名2：值2
//      响应头名3：值3
//空行 （必须有）
//响应体 客户端返回的主要内容
```

#### AJAX（异步JavaScript和 XML）
- AJAK的核心思想：再不重新加载整个网页的情况下，通过JavaScript在后台与服务器交换数据并更新部分网页内容
- 异步性：当向服务器发送请求后，JavaScript不会等待响应，而是继续执行后续代码。当响应就绪时，再通过回调函数处理数据。
- 数据格式： AJAX数据交换主要格式已由XML转为JSON
#### 工作原理
事件发生（由用户触发）->JavaScript创建一个请求对象->通过设置的请求方法配置请求->将请求发送到服务器->由客户端处理响应

### ES6-ES11
#### ES6
- let和const<br>块级作用域和变量声明：使用let声明变量，const声明常量，解决了var的变量提升能力
- 箭头函数<br>一种新的简洁的函数语法，箭头函数提供更简洁的函数语法，但是，箭头函数不绑定自己的this，如果在箭头函数中调用this则会得到整个窗口体的对象
- 模板字符串<br>模板字符串增强了字符串的功能使用反引号（``）创建字符串，支持多行文本和变量的嵌入
- 解构赋值<br>结构赋值语法可以将数组中的值或对象的属性提取到不同的变量当中
```javascript
// 数组解构
const arr = [1, 2, 3];
const [a, b, c] = arr;
console.log(a, b, c); // 1 2 3

// 对象解构
const person = { name: "李四", age: 30, city: "北京" };
const { name, age } = person;
console.log(name, age); // 李四 30

// 函数参数解构
function printPerson({ name, age }) {
  console.log(`${name}今年${age}岁`);
}
printPerson(person);
```
- 默认参数<br>ES6允许为函数的参数设置默认值，当用户没有传递参数时，参数会自动被设置成默认值
#### ES7
- ES7提供了includes()方法用于判断是否包含某元素，还提供指数运算符
#### ES8-ES11
- async/await<br>异步函数，允许使用同步的代码风格，通过async关键字声明异步函数，通过await关键字等待异步操作完成，await关键字只能用在async函数中,这两个关键字让异步代码看起来像同步代码，提高了代码的可读性
- Promise<br>Promise对象用于处理异步操作，Promise对象是一个容器，里面保存着某个未来才会结束的事件（通常是一个异步操作），Promise对象有两个状态：pending（进行中）和fulfilled（已成功）
- ?.可选链操作符
