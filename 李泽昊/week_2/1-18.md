### 1，JavaScript异步编程
#### JavaScript是单线程语言，一次只能执行一个任务为避免阻碍主线程进程因此需要异步编程
#### 异步编程的优势
* 非阻塞性：异步处理不会阻塞主线程
* 提升性能：可以从分利用系统资源
* 提升用户体验：避免由于同步请求导致的页面卡顿
* 并发处理：可以同时处理多个任务
#### 异步回调
```javascript
// 模拟异步操作
function fetchData(callback) {
  console.log('1');
  
  setTimeout(() => {
    const data = 2
    callback(data);
  }, 1000);
  console.log('3');
}

// 使用回调
fetchData((data) => {
  console.log(data);
});
// 输出结果：1 3 2
```
#### Promise对象
#### Promise是ES6引入的异步编程解决方案，代表一个异步操作的完成或失败
#### 三种状态
* Pending（进行中）：初始状态
* Fulfilled（已成功）：操作成功完成
* Rejected（已失败）：操作失败
```javascript
const promise = new Promise((resolve, reject) => {
  // 异步操作
  if (true) {
    resolve(result);
  } else {
    reject(error);
  }
});

promise
  .then(result => console.log(result))
  .catch(error => console.error(error));

```
#### async/await

基本概念:async/await 是 ES2017 引入的语法糖，让异步代码看起来像同步代码。

#### 并发执行
```javascript
// 串行执行（慢）
async function serial() {
  console.log('开始');
  
  const result1 = await task1();
  console.log('任务1完成');
  
  const result2 = await task2();
  console.log('任务2完成');
  
  const result3 = await task3();
  console.log('任务3完成');
  
  return [result1, result2, result3];
}

// 并行执行（快）
async function parallel() {
  console.log('开始');
  
  const [result1, result2, result3] = await Promise.all([
    task1(),
    task2(),
    task3()
  ]);
  
  console.log('所有任务完成');
  return [result1, result2, result3];
}

// 独立任务并发
async function independentTasks() {
  const task1Promise = task1();
  const task2Promise = task2();
  const task3Promise = task3();
  
  const result1 = await task1Promise;
  const result2 = await task2Promise;
  const result3 = await task3Promise;
  
  return [result1, result2, result3];
}
```
### 2，JavaScript模块化
模块化是将代码分割成独立，可复用的单元，每个模块都有自己的作用域，只需要提供必要的接口
#### JavaScript 模块化方案

1. CommonJS (Node.js)CommonJS 是 Node.js 使用的模块化规范，使用 `require()` 导入，`module.exports` 或 `exports` 导出。
2. ES Modules (ES6，浏览器和 Node.js)ES Modules 是 ES6 引入的原生模块化方案，使用 `import` 和 `export` 关键字，支持浏览器和现代 Node.js。
3. AMD (RequireJS，主要用于浏览器）
4. UMD (通用模块定义，兼容 CommonJS 和 AMD）
